# CSS-Bot — Бот-помощник для первокурсников ВСН (ОП "Вычислительные социальные науки")

# Telegram-бот, созданный для помощи первокурсникам образовательной программы "Вычислительные социальные науки" Высшей школы экономики.

Бот помогает быстро найти учебных ассистентов, записи лекций, места для учёбы, полезные ссылки и просто поддерживает в трудную минуту.
(Реализован в рамках дисциплины "Программирование на Python" как проектное задание)
---

## Возможности

### Учебные ассистенты
Бот поможет найти ассистента по следующим предметам:
- Линейная алгебра и геометрия
- Программирование на Python
- Математический анализ
- Программирование на C++
- Алгоритмы и структуры данных
- Дискретная математика

### Записи лекций
Доступны записи лекций по:
- **Питону** — 12 лекций с Яндекс.Диска
- **Линейной алгебре** — ссылки на доступные записи и Google Drive

### Где поботать?
Бот расскажет о лучших коворкинг-пространствах в двух кампусах:
- **Покровка** — библиотека, Большой атриум, кафе «Груша»
- **Мясницкая (М20)** — коворки на этажах, Малый атриум, коворк на Кривоколенном

### Полезные ссылки
Быстрый доступ к:
- Материалам по ЛинАлу
- Wiki ФКН по Python
- Ведомости по Питону и ОРГ

### Хочется плакать
Раздел психологической поддержки с советами по 10 типичным проблемам первокурсника:
- Не понимаю лекции по Питону/ЛинАлу
- Горят дедлайны
- Не хватает сил
- Нужен кофеин
- Хочется спать
- Ненавижу БЖД
- ...и другие студенческие беды 

###  Хочу в бар
Список баров на Чистых прудах для культурного отдыха.
